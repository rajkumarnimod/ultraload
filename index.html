UltraLoad.js Documentation

Introduction

UltraLoad.js is a versatile preloader library designed to enhance user experience during page or content loading. With 50+ loading animations and 30+ text effects, it's perfect for any web project.

Installation

Via NPM

npm install ultraload-js

Via CDN

<!-- CSS -->
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/ultraload-js/dist/ultraload.min.css">

<!-- JS -->
<script src="https://cdn.jsdelivr.net/npm/ultraload-js/dist/ultraload.umd.min.js"></script>
